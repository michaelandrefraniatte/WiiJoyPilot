WiiJoyPilot

Be Like A Pro

Using Joycon Left and Wiimote as Xbox Controller in FPS Games

The program allow to use Wiimote and Joycon left in FPS games with easy Configurations. It change in the game as soon as you change Configurations in the program. It's easy to understand and easy to win with this program. You can play all your FPS games the way you want if the Xbox controller is well supported in. Let's see...

Considerations About the Program

Theres's a driver of old generation to install manually as old generic driver from peripheral manager under control panel (choosing the file with .inf extension). And also, another driver to reorder All your Physical and Virtual Controllers from joy.cpl (Win + R). After the program start, you shall put Joycon left and Wiimote in pairing mode, by pressing the buttons to connect it via Bluetooth. The connection is automatic with just that, and the disconnection is totally automatic after you close the program. If you are interested in other programs like this one, check the presentation of supporting controllers.
